# Autonomous Apple Harvesting Robot

## Components Used

### Hardware
1. **Robotic Chassis**
2. **Arduino Uno**
3. **L293D Motor Driver**
4. **Ultrasonic Sensor**
5. **HC-05 Bluetooth Module**
6. **Servo Motor**
7. **Stepper Motor**
8. **RGB Color Sensor**

## Software Used
- **Arduino Programming**

### Arduino Code
1. **Line Following**
2. **Obstacle Avoiding**
3. **Interfacing with Servo**
4. **Interfacing with Stepper**

## Applicaion
- Line Following
- Obstacle Avoiding
